# continue

continue ek python mai ek *keyword* hai..


![Mobile Video Recorder](assets/how-continue-statement-works.jpg)

```python
counter = 0
string = "navgurukul"
while (counter < len(string)):
    counter += 1

    if string[counter] == "a":
        continue

    if string[counter] == "u":
        continue
    
    print(string[counter])

print("The end", string[counter])
 ```
