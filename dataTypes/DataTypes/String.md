### String data type

Python string characters ya text ka sequence hota hai.ye textual data ko handle krta hai.hum quotes(single ya double quote) me jo bhi likhte hai wo string khelata hai.

**Example :-**

```python
a= "Hello python"
b="Hello world"
print(a)
print(b)
 ```
**Output :-**

`Hello python`

`Hello world`

```python
a="hello"

 ```
